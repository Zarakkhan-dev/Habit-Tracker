The Habit Tracker is a Python-based application that allows users to register, login, add habits, track habit completions, and view habit progress.

Instructions to Run:
Prerequisites:
Ensure Python 3.x is installed on your system.
Steps:
Open Terminal (Command Prompt):

Open your terminal or command prompt.
Navigate to Directory:

Use cd command to navigate to the directory where habit_tracker.py is located.
Run the Program:

Type python habit_tracker.py and press Enter.
Interact with the Application:

Follow the on-screen instructions:
Choose option 1 for registration or 2 for login.
Provide username and password as prompted.
Add habits, track completions, find habits, and display progress as per the menu options.
Exiting the Application:

To exit the application, select option e or complete the actions and choose the exit option.
Usage Notes:
The application is text-based and operates in the terminal or command prompt.
Each action (registration, login, habit management) requires user input following the displayed prompts.
